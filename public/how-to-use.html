How to Use STREAMNEXUS

1

Get Started

Click the "Get Started" button on the home page to access the streaming dashboard.

2

Choose Your Option

Select "Start Class" to host a new class or "Join Class" to enter an existing class.

3

Share Your Class Code

If starting a class, share the generated class code with your students. The code appears at the top of your stream.

4

Join a Class

If joining a class, enter the class code provided by your instructor to access the live session.

5

Use Stream Controls

Use the camera and microphone controls at the bottom of the stream to manage your audio/video. Toggle fullscreen mode for a better viewing experience.
